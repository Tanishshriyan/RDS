Detection Pipeline

A multi-stage detection pipeline that combines sensors, enrichment, scoring, and automated response.

1

Sensors

psutil streams process metrics; watchdog monitors file events; disk and network samplers detect abnormal I/O patterns.

2

Enrichment

Events are enriched with entropy calculations, suspicious name detection, and behavioral fingerprinting.

3

Scoring

Conservative algorithms blend signals into a 0-100 suspicion score, avoiding false positives.

4

Response

High-score events trigger kill-switch, broadcast to dashboard, and persist for forensic analysis.
